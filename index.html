Visual studio 설치 및 실행방법 C언어의 소스파일 확장자는 .c가 되어야 한다. 기본적으로 뜨는 .cpp는 C++문법으로 컴파일됨 단축키 Ctrl + F5 : 실행하기 C언어로 구현된 모든 프로그램에는 main이라는 이름의 함수가 정의되어야 함 #include //헤더파일 stdio.h에는 printf 함수의 호출에 필요한 정보가 존재 int main(void) //출력형태가 int이고 입력의 형태가 void인 main이라는 이름의 함수 return 0; // 함수 종료 또는 함수를 호출한 영역으로 값을 전달(반환) 의미 가짐 3. 변수란 무엇인가 변수란 값을 저장할 수 있는 메모리 공간에 붙은 이름 int num; //num이라는 이름의 변수 선언 변수를 선언하고 꼭 초기화를 해야한다. int num1 = 20; //변수1선언 및 초기화 4. 기본 자료형(Primitive) 자료형은 프로그램에서 선언된 변수들이 기억 공간에서 어떻게 저장되고 처리되어야 할지 컴파일러에 알려준다. 1) 정수형 부호의 유무에 따라 signed / unsigned 변수의 크기에 따라char(1바이트) / short(2) / int(4) / long(4) 2) 실수형 변수의 크기에 따라 float(4바이트) / double(8) 5. 문자를 입력 받는 방법 1) 문자 입출력 함수 : 문자 1개를 입력 받아 출력 getchar() : 문자입력 putchar() : 문자출력 char Letter; // Letter이라는 변수 선언 Letter = getchar(); //문자 1개를 입력받아 Letter에 대입 putchar(Letter); //Letter 출력 2) 문자열 입출력 함수 : 문자 여러 개를 입력 받아 출력 gets() : 문자열입력 puts() : 문자열출력 char Letter; //Letter이라는 변수 선언 gets(Letter); //문자열을 입력 puts(Letter); //문자열 출력 6. 조건문 1)if문 if(조건) 2)if ~ else 문 if절의 조건이 참이 아니면 else가 실행 3) if ~ else if ~ else문 if(조건1) ~ else if(조건2) ~ else 조건이 여러 개이면 else if 여러 개 사용 가능, else는 위의 조건들이 불만족일 때 7. 반복문 1) while문 : while(조건) 2) do ~ while문 반복조건을 뒷부분에서 검사 반복영역을 최소한 한번은 실행하는 구조 3) for문 for(초기식 ; 조건식 ; 증감식) 4) break 반복문을 탈출할 때 사용 5) continue 반복문 안에서 실행중인 위치에 상관없이 반복문의 조건검사 위치로 이동 8. 배열 배열이란 다수의 데이터를 저장하고 처리하는 경우에 유용하게 사용할 수 있는 것 int Name[4]; //길이가 4인 int형 배열 선언 , 배열 이름은 Name Name[0] = 10; //배열 Name의 첫 번째 요소에 10 저장, 배열은 0번째부터 시작 Name[3] = 40; //배열 Name의 네 번째 요소에 40저장
